# Incubyte COE Monorepo

This monorepo consolidates all work completed during the Incubyte Center of Excellence (COE) onboarding program — a structured 10-day curriculum covering TDD, GraphQL, Docker, Elasticsearch, Redis, AI tooling, and OOP principles.

## Repository Structure

| Folder | Description |
|---|---|
| [`string-calculator`](./string-calculator) | TDD String Calculator kata (Day 1) |
| [`incubyte-culture-and-tdd-foundations`](./incubyte-culture-and-tdd-foundations) | Daily learning notes — culture, concepts, and reflections (Day 1–10) |
| [`graphql-learning-backend`](./graphql-learning-backend) | Ruby on Rails API-only app with GraphQL, RSpec, Docker, Elasticsearch, Redis |
| [`graphql-learning-frontend`](./graphql-learning-frontend) | React + TypeScript + Apollo Client frontend with Chakra UI and Tailwind CSS |
