# AI Plugins Overview

## Bee

* Bee is an AI plugin that promotes spec-driven development and software engineering best practices.
* It guides developers through the appropriate workflow based on the size and risk of a task.
* For larger features, it can help with specification, architecture, implementation, testing, verification, and code review.
* The developer remains in control while Bee acts as a guide throughout the development process.

## Learn

* Learn is an AI plugin that teaches technologies through hands-on projects instead of theory alone.
* It provides a project-based learning path and guides the developer step by step.
* The developer writes all the code, while the AI acts as a mentor rather than generating the complete solution.
* It can track learning progress and adapt the pace based on the learner's experience.
