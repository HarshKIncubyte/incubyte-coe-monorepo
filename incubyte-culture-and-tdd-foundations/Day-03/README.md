# Day 3 – Testing GraphQL with RSpec, VCR & Strong Migrations

> Part of my Incubyte COE Learning Journey

## Overview

Extended the GraphQL API from Day 2 with comprehensive test coverage using
RSpec. Integrated VCR for external API testing and added strong_migrations

- GraphQL resolver and mutation tests
- 80%+ test coverage on GraphQL layer

- Strong Migrations to prevent unsafe migrations
- Safe index creation with algorithm: :concurrently
