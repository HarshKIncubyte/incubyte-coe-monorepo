# Day 4 - Frontend with React & Apollo Client

## What I built

A React + TypeScript frontend that connects to the GraphQL backend built in Day 2/3.

## Repo

[graphql-learning-frontend](https://github.com/HarshKIncubyte/graphql-learning-frontend)

## Topics covered

- React + TypeScript app with Vite
- Apollo Client — useQuery and useMutation
- Context API — shared state across components
- Jest + React Testing Library
- MockedProvider for GraphQL API mock tests

## Connected to

- Backend: [graphql-learning](https://github.com/HarshKIncubyte/graphql-learning) (Day 2/3)
- Frontend calls GraphQL API at `http://localhost:3000/graphql`
